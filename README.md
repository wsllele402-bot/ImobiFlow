# ImobiFlow - Gestão de Aluguéis

Uma plataforma completa e profissional para imobiliárias e proprietários gerenciarem múltiplos imóveis, inquilinos, cobranças e relatórios de fechamento.

## 🚀 Funcionalidades

- **Dashboard Financeiro:** Visão geral de entradas, saídas e saldo para repasse.
- **Gestão de Imóveis:** Cadastro e monitoramento de kitnets, casas, apartamentos e galpões.
- **Gestão de Inquilinos:** Controle de contratos, documentos e pagamentos.
- **Relatórios de Fechamento:** Geração de PDFs profissionais com detalhamento por imóvel e extratos completos.
- **Integração Supabase:** Backend robusto para persistência de dados.
- **Cobranças Automatizadas:** Preparado para integração com gateways de pagamento (Asaas).

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React 19, TypeScript, Tailwind CSS 4.
- **Animações:** Framer Motion.
- **Gráficos:** Recharts.
- **PDF:** jsPDF e html2canvas.
- **Backend/DB:** Supabase.
- **Ícones:** Lucide React e FontAwesome.

## 📦 Como Rodar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone <url-do-seu-repositorio>
   cd imobiflow
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env` baseado no `.env.example` e preencha as credenciais do seu Supabase.

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Acesse no navegador:**
   `http://localhost:3000`
